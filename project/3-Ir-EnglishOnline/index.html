Ir-EnglishOnline

Ir-EnglishOnline is an online english academy. The Council aimed to make english learning possible for all it's users.
It was very important for them to have a flexible website which let them create new sections once in a while. We concluded to develope a website with a user management system and an special content management system which I designed for them to cover all their training oriented requeirements.
The website was designed and developed in 2011.

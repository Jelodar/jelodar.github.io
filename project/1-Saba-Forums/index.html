Saba Forums

Saba is a Open Source Forum Management System which has many features and is highly flexible.
I started the development of Saba in 2005 with the first release was in 2006. The latest release of Saba is v3.0.0 which was released in July 2012.
The source code is available on SourceForge website.


Some features:

  • Support for Multiple Languages
  • Plugins
  • Themes
  • High Flexibility
  • User Groups
  • Permissions System

Academy of Medical Sciences

Academy of Medical Sciences of Iran's website was previously powered by a CMS (e107). When they came to me they had some problems with their website. The main problems was about security issues and lack of flexibility.
They needed to have a website with their own rules and access levels for different user types. I developed the backend using PHP & MySQL with my own designed framework, and Front-end using HTML5, CSS3, and jQuery.
I also configured their Free BSD server which the apache webserver was running on, so then it became more secure and managable.
The project was done in 2013.
